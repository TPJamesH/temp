2. Typography

Primary Font: Roboto, for headlines and navigation.

Secondary Font: Open Sans, for body copy and UI elements.

Monospace: Source Code Pro, for code snippets and technical text.

3. Logo Usage

Always maintain clear space around the logo equal to the height of the “A” in Acme. Do not stretch, rotate, or recolor the logomark.

4. Iconography

Icons should be simple, geometric, and use the primary brand color. Use 24×24 px grid.
